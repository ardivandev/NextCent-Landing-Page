# Nexcent Landing Page

A responsive landing page inspired by the Nexcent design, built using HTML, Tailwind CSS v4, and JavaScript.

## ✨ Features

- Responsive Design
- Mobile Navigation Menu
- Smooth Scroll Animations
- Modern UI Layout

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS v4
- JavaScript
- AOS

## 🚀 Getting Started

```bash
npm install
npm run dev
```
